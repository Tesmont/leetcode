A small archive of solved LeetCode problems, written in C# and organized to stay tidy over time.
For now the primary language is C#, but other languages may appear here in the future as well.

What lives here:

- main solutions for each problem
- alternative implementations like `BinarySearch` or `DynamicProgramming`
- xUnit tests for every added problem

```text
src      -> solution code
tests    -> mirrored test coverage
problem  -> one directory, one clear entry point
```

No framework drama.
No extra layers.
Just problems, approaches, and passing tests.
